## Featured Projects

### 🏗️ Scalable AWS E-Commerce Platform
A cloud-native e-commerce infrastructure built on AWS, designed for high availability, security, and automated delivery. The architecture leverages EC2 Auto Scaling with an Application Load Balancer, CloudFront for content distribution, RDS Multi-AZ for database resilience, and ECS for containerized deployments. Secrets are managed through AWS Secrets Manager, and CI/CD is handled via GitHub Actions.

**Tech:** AWS VPC · EC2 Auto Scaling · Application Load Balancer · CloudFront · RDS Multi-AZ · ECS · Secrets Manager · IAM · Docker · GitHub Actions

### ⚙️ Distributed E-Commerce Microservices Platform
A distributed microservices system for order management, built around event-driven communication and the Saga pattern for distributed transaction consistency. Services communicate asynchronously via Apache Kafka, with Redis used for caching and Eureka for service discovery. Designed to support independent scaling and reliable order processing workflows.

**Tech:** Java · Spring Boot · Apache Kafka · Eureka · API Gateway · Redis

### 🚗 CabConnect — Full-Stack Ride Management Platform
A full-stack ride management application covering ride booking, trip tracking, and user profile management. The backend is built with Spring Boot and secured using Spring Security with JWT authentication. The frontend uses React.js and Next.js with Redux Toolkit for state management and Tailwind CSS for responsive UI design, backed by PostgreSQL.

**Tech:** React.js · Next.js · Spring Boot · PostgreSQL · Spring Security · JWT · Redux Toolkit · Tailwind CSS
